# **App Name**: ShopStack

## Core Features:

- User Authentication: Secure user registration and login with JWT authentication.
- Product Listing and Filtering: Display products with advanced filtering options.
- Shopping Cart Management: Add, remove, and update items in the shopping cart.
- Order Placement: Allow users to place orders, including payment processing.
- Payment Integration: Integration with Stripe and Razorpay payment gateways.
- Admin Product Management: Enable admins to add, update, and delete products.
- AI Product Recommendations: Suggest related products and personalize the shopping experience based on browsing history. LLM analyzes past product views and purchase behavior. This feature should function as a tool, only engaging under specifically understood scenarios to improve relevance and avoid overwhelming the user with unwanted notifications.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) for a modern and luxurious feel.
- Background color: Light grey (#F2F0F7), creating a clean and spacious backdrop.
- Accent color: Soft lavender (#D0BCFF), providing subtle contrast and highlighting key elements.
- Body font: 'Inter', a sans-serif font with a modern and neutral look for readability.
- Headline font: 'Space Grotesk', a geometric sans-serif, used for impactful headings.
- Use clear and simple icons for navigation and product categories.
- Implement a responsive grid layout for consistent presentation across devices.
- Subtle animations for transitions and interactions to enhance user experience.
